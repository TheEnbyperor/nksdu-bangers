# Neko Desu Bangers

A Django web app for certifying songs as bangers.
Available at [nkdsu-bangers.magicalcodewit.ch](https://nkdsu-bangers.magicalcodewit.ch).

Data is provided by the [nkdu.su](https://nkd.su/) voting site API.
